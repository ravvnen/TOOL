# TOOL
*Team-Governed AI Memory for Coding Assistants*

## Vision
Reduce “team AI drift” by promoting/retracting trusted knowledge from PR/CI events and pushing deltas to each developer’s assistant with provenance and replay.

## Structure
- `/thesis` – LaTeX sources
- `/poc` – prototypes & results
- `/docs` – notes & related work
- `/figures` – diagrams, exports
